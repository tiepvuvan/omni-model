---
title: Swift (iOS)
description: Call omni-model from an iOS app — via Firebase callable functions or a self-hosted HTTP proxy.
---

There are two ways an iOS app reaches omni-model. **Serverless** calls a Firebase Callable
Function; the Firebase SDKs attach Firebase Auth + App Check tokens for you. **Self-hosted** uses
the real [MacPaw/OpenAI](https://github.com/MacPaw/OpenAI) client plus a small drop-in
`OpenAIMiddleware` — [`examples/ios/OmniModelClient.swift`](https://github.com/tiepvuvan/omni-model/blob/main/examples/ios/OmniModelClient.swift) —
that points it at your proxy and attaches the right auth header (custom JWT, Firebase Auth, App
Check, App Attest, or DeviceCheck) on every request, streaming included.

Both speak the OpenAI wire format: requests carry `model` and `messages`, responses carry
`choices`, and streaming emits `chat.completion.chunk` objects.

<Tabs>
  <Tab title="Serverless (Firebase)">
    Your app calls the callable `ext-omni-model-proxy-chat` deployed by the
    [Firebase Extension](/installation/firebase). No API keys or proxy URL live in the binary — the
    Firebase client SDK forwards the signed-in user's Auth ID token and an App Check token on every
    call.

    <Steps>
      <Step title="Add the SDKs and initialize App Check + Auth">
        Add `FirebaseFunctions`, `FirebaseAppCheck`, and `FirebaseAuth` (Swift Package Manager or
        CocoaPods). Install the App Check provider **before** `FirebaseApp.configure()` so tokens
        are minted from the start. On device, use the [App Attest](/security/app-attest) provider.

        ```swift
        import FirebaseCore
        import FirebaseAppCheck
        import FirebaseAuth

        final class OmniAppCheckFactory: NSObject, AppCheckProviderFactory {
          func createProvider(with app: FirebaseApp) -> AppCheckProvider? {
            // App Attest on iOS 14+; DeviceCheck is the fallback for older devices.
            if #available(iOS 14.0, *) {
              return AppAttestProvider(app: app)
            }
            return DeviceCheckProvider(app: app)
          }
        }

        // In application(_:didFinishLaunchingWithOptions:)
        AppCheck.setAppCheckProviderFactory(OmniAppCheckFactory())
        FirebaseApp.configure()
        ```

        The proxy defaults to `REQUIRE_AUTH=true`, so make sure a user is signed in (e.g.
        `Auth.auth().signInAnonymously()`) before calling. See
        [Installation → Firebase](/installation/firebase) for the extension parameters.
      </Step>

      <Step title="Non-streaming call">
        ```swift
        import FirebaseFunctions

        let functions = Functions.functions()

        func complete(_ prompt: String) async throws -> String {
          let callable = functions.httpsCallable("ext-omni-model-proxy-chat")
          let result = try await callable.call([
            "model": "gpt-4o-mini",
            "messages": [["role": "user", "content": prompt]],
          ])
          // result.data is the OpenAI ChatCompletion as [String: Any].
          let data = result.data as? [String: Any]
          let choices = data?["choices"] as? [[String: Any]]
          let message = choices?.first?["message"] as? [String: Any]
          return message?["content"] as? String ?? ""
        }
        ```
      </Step>

      <Step title="Streaming call">
        The callable streaming API yields `.message` chunks (each a `chat.completion.chunk`) and a
        final `.result` (the aggregated `ChatCompletion`).

        ```swift
        import FirebaseFunctions

        func stream(_ prompt: String, onDelta: @escaping (String) -> Void) async throws {
          let callable = Functions.functions().httpsCallable("ext-omni-model-proxy-chat")
          let stream = try callable.stream([
            "model": "gpt-4o-mini",
            "stream": true,
            "messages": [["role": "user", "content": prompt]],
          ])

          for try await response in stream {
            switch response {
            case .message(let chunk):
              // chunk.data is a chat.completion.chunk: choices[0].delta.content
              let data = chunk.data as? [String: Any]
              let choices = data?["choices"] as? [[String: Any]]
              let delta = choices?.first?["delta"] as? [String: Any]
              if let piece = delta?["content"] as? String { onDelta(piece) }
            case .result(let final):
              _ = final.data // aggregated ChatCompletion, if you need usage/finish_reason
            @unknown default:
              break
            }
          }
        }
        ```

        <Note>
          The callable `.stream(...)` API requires FirebaseFunctions 11.4+ and a 2nd-gen callable
          (the Extension deploys one). On older SDKs, use the non-streaming call above.
        </Note>
      </Step>

      <Step title="Handle errors">
        Failures surface as `NSError` in `FunctionsErrorDomain`; read the
        `FunctionsErrorCode`.

        ```swift
        do {
          let text = try await complete("Hello")
        } catch let error as NSError {
          if error.domain == FunctionsErrorDomain,
             let code = FunctionsErrorCode(rawValue: error.code) {
            switch code {
            case .unauthenticated:    break // no signed-in user — prompt sign-in
            case .failedPrecondition: break // App Check failed — device not attested
            case .resourceExhausted:  break // rate limit / token budget — back off
            case .invalidArgument:    break // malformed request body
            default:                  break
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>

  <Tab title="Self-hosted proxy (MacPaw/OpenAI)">
    You use the real [MacPaw/OpenAI](https://github.com/MacPaw/OpenAI) `OpenAI` client — its full
    API, its types, its updates. The drop-in
    [`OmniModelClient.swift`](https://github.com/tiepvuvan/omni-model/blob/main/examples/ios/OmniModelClient.swift)
    adds only a custom **`OpenAIMiddleware`** that points it at your proxy and stamps the caller's
    identity onto every request — streaming and non-streaming alike. Nothing forwards
    `OpenAIProtocol` methods, so there's no wrapper to maintain.

    <Steps>
      <Step title="Add MacPaw/OpenAI">
        In Xcode, **File → Add Package Dependencies…** and add
        `https://github.com/MacPaw/OpenAI` (Up to Next Major Version, from `0.4.0`).
      </Step>

      <Step title="Copy in the middleware and set your endpoint">
        Copy [`examples/ios/OmniModelClient.swift`](https://github.com/tiepvuvan/omni-model/blob/main/examples/ios/OmniModelClient.swift)
        into your app target and edit one line:

        ```swift
        // OmniModelClient.swift  — >>> EDIT ME <<<
        public static let production = OmniEndpoint("https://your-proxy.example.com")
        ```

        It provides `OmniModel.makeOpenAI(box:)` (returns a real MacPaw `OpenAI`), an `OmniAuthBox`
        (a thread-safe header holder), and an `OmniAuthProvider` for each auth mode. Firebase
        providers compile only when the Firebase SDKs are present.
      </Step>

      <Step title="Pick the auth provider that matches your proxy">
        Choose the provider matching a [`security` verifier](/security/custom-jwt) in your config.

        <AccordionGroup>
          <Accordion title="Custom JWT / Supabase — Authorization: Bearer">
            Matches the [`jwt` / `supabase`](/security/custom-jwt) verifier.

            ```swift
            let auth = BearerTokenAuth { try await myAuth.freshAccessToken() }
            ```
          </Accordion>
          <Accordion title="Firebase Auth — Authorization: Bearer (ID token)">
            Matches the [`firebase-auth`](/security/custom-jwt) verifier. Needs the `FirebaseAuth`
            SDK and a signed-in user.

            ```swift
            let auth = FirebaseIDTokenAuth()
            ```
          </Accordion>
          <Accordion title="Firebase App Check — X-Firebase-AppCheck">
            Matches the [`firebase-app-check`](/security/firebase-app-check) verifier. Needs the
            `FirebaseAppCheck` SDK (initialize the provider before `FirebaseApp.configure()`).

            ```swift
            let auth = FirebaseAppCheckAuth()
            ```
          </Accordion>
          <Accordion title="Apple App Attest — x-appattest-* headers">
            Matches the [`apple-app-attest`](/security/app-attest) verifier. Each refresh runs the
            challenge/assert flow against your proxy's `/auth/app-attest/*` routes (the one-time
            attest/register handshake happens on first use).

            ```swift
            let auth = AppAttestAuth(endpoint: .production)
            ```
          </Accordion>
          <Accordion title="Apple DeviceCheck — X-Apple-Device-Token">
            Matches the [`apple-device-check`](/security/app-attest) verifier.

            ```swift
            let auth = DeviceCheckAuth()
            ```
          </Accordion>
        </AccordionGroup>
      </Step>

      <Step title="Make requests with the real OpenAI client">
        Build the client once, then `refresh` the box before a call so the middleware injects a
        fresh token. `model` is a plain string, so pass whatever your
        [routing](/model-routing/routing) expects — a real model id or an alias like `"smart"`.

        ```swift
        import OpenAI

        let box = OmniAuthBox()
        let openAI = OmniModel.makeOpenAI(box: box)   // a genuine MacPaw OpenAI

        // Non-streaming
        try await box.refresh(from: auth)
        let result = try await openAI.chats(
          query: ChatQuery(messages: [.user(.init(content: .string("Hello!")))], model: "gpt-4o-mini")
        )
        print(result.choices.first?.message.content ?? "")

        // Streaming — the middleware injects auth on the stream too
        try await box.refresh(from: auth)
        for try await chunk in openAI.chatsStream(
          query: ChatQuery(messages: [.user(.init(content: .string("Write a haiku.")))], model: "gpt-4o-mini")
        ) {
          for choice in chunk.choices {
            if let piece = choice.delta.content { print(piece, terminator: "") }
          }
        }

        // Embeddings
        try await box.refresh(from: auth)
        let vectors = try await openAI.embeddings(
          query: EmbeddingsQuery(model: "text-embedding-3-small", input: "hello world")
        )
        ```

        <Note>
          Why `refresh` before each call? MacPaw's middleware runs **synchronously**, so it can't
          `await` a token itself — `refresh` does the async work and the middleware reads the result.
          For **App Attest** this is required per request (its assertion is single-use); for JWT /
          App Check the call is cheap and you can refresh on a schedule instead. Everything else —
          `images`, `audio`, `models`, retries, error types — is stock MacPaw/OpenAI.
        </Note>
      </Step>
    </Steps>

    <Accordion title="No dependencies? Raw URLSession">
      You can skip MacPaw/OpenAI and hit the endpoint directly — the body and responses are exactly
      OpenAI's. Send an `Authorization: Bearer <token>` (or the header your verifier expects).

      ```swift
      func complete(_ prompt: String, token: String) async throws -> String {
        var request = URLRequest(url: URL(string: "https://your-proxy.example.com/v1/chat/completions")!)
        request.httpMethod = "POST"
        request.setValue("application/json", forHTTPHeaderField: "Content-Type")
        request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization")
        request.httpBody = try JSONSerialization.data(withJSONObject: [
          "model": "gpt-4o-mini",
          "messages": [["role": "user", "content": prompt]],
        ])

        let (data, response) = try await URLSession.shared.data(for: request)
        guard let http = response as? HTTPURLResponse, http.statusCode == 200 else {
          // Body is an OpenAI-style { "error": { message, type, param, code } }.
          throw URLError(.badServerResponse)
        }
        let json = try JSONSerialization.jsonObject(with: data) as? [String: Any]
        let choices = json?["choices"] as? [[String: Any]]
        let message = choices?.first?["message"] as? [String: Any]
        return message?["content"] as? String ?? ""
      }
      ```

      For streaming, set `"stream": true`, read `URLSession.shared.bytes(for:)` line by line, strip
      the `data: ` prefix, stop at `data: [DONE]`, and decode each line as a `chat.completion.chunk`.
    </Accordion>
  </Tab>

  [`swift/OmniModelClientKit`](https://github.com/tiepvuvan/omni-model/tree/main/swift/OmniModelClientKit)

</Note>
